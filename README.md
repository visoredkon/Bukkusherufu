## Installation

1. Clone the repository:

   ```
   git clone https://github.com/visoredkon/bukkusherufu-api.git
   ```

2. Navigate to the project directory:

   ```
   cd bukkusherufu-api
   ```

3. Install the dependencies:

   ```
   npm install
   ```
